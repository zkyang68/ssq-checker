# 双色球开奖号码查询

对照号码 **04 07 08 11 12 28 | 06**，每期自动比对中奖结果。

## 功能

- 自动获取最新双色球开奖数据（由 GitHub Action 每日更新）
- 对照用户号码，显示每期中奖等级和奖金
- 统计累计投入、中奖次数、净收益
- 从 2026-05-12 起追踪所有购买数据

## 中奖规则

| 等级 | 条件 | 奖金 |
|------|------|------|
| 一等奖 | 6红+1蓝 | 浮动 ~500万+ |
| 二等奖 | 6红+0蓝 | 浮动 ~20万+ |
| 三等奖 | 5红+1蓝 | 3,000元 |
| 四等奖 | 5红+0蓝 或 4红+1蓝 | 200元 |
| 五等奖 | 4红+0蓝 或 3红+1蓝 | 10元 |
| 六等奖 | 0~2红+1蓝 | 5元 |

## 技术

- 纯静态 HTML，托管于 GitHub Pages
- 数据源：500.com（GitHub Action 每二/四/日晚自动抓取）
- 本地存 localStorage 记录追踪数据
